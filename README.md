# Weather App

Displays a 5-day weather forecast, where each day shows the high and low temperatures, and an image for sunny/rainy/cloudy/snowy.

## Technologies
1. React
2. Typescript
3. React Router
4. React Hooks
5. Axios
6. Atomic Design Pattern

[Source](https://weather.com/weather/today/l/47.06,28.85?par=google&temp=c)
